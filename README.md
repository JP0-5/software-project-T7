# Blackjack Fever
Blackjack Fever is an online web-based multiplayer blackjack game with a twist that we implemented to add more player interaction.

We aimed to take a classic card game and adapt it to the modern gaming landscape.

[Play the game here.](https://blackjack.eu.pythonanywhere.com/)

[Read the documentation here.](https://github.com/JP0-5/software-project-T7/blob/main/doc/documentation.md)
